Introduction

In Windows Store apps is possible to use a WebView control to show web page. Sometimes is need to intercept the events from the web page and get them, in the XAML page that contais the Webview, to do others task based in the action.

There are some changes from Windows 8.0 to Windows 8.1, that can be find in a old article What’s new in WebView in Windows 8.1 and there is an interesting sample How to intercept JavaScript alert in WebView in universal Windows apps from OneCode Team. In this sample, we will see a solution for when we click in a button.


Building the Sample

You only need Visual Studio 2013 and Windows 8.1.

Description

 and in code be hide, define the WebView's NavigationCompleted event as following:
C#
Edit|Remove
csharp 
  private async void WebView_OnNavigationCompleted(WebView sender, WebViewNavigationCompletedEventArgs args)
        {
            await WebView.InvokeScriptAsync("eval", new[]
            {
                "var signButton=document.getElementById(\"MyButton\");" +
                "if (signButton.addEventListener) {" +
                    "signButton.addEventListener(\"click\", MyButtonClicked, false);" +
                "} else {" +
                    "signButton.attachEvent('onclick', MyButtonClicked);" +
                "}  " +
                "function MyButtonClicked(){" +
                    " window.external.notify('%%' + location.href);"+
                 "}"
            });
        }
  private async void WebView_OnNavigationCompleted(WebView sender, WebViewNavigationCompletedEventArgs args) 
        { 
            await WebView.InvokeScriptAsync("eval", new[] 
            { 
                "var signButton=document.getElementById(\"MyButton\");" + 
                "if (signButton.addEventListener) {" + 
                    "signButton.addEventListener(\"click\", MyButtonClicked, false);" + 
                "} else {" + 
                    "signButton.attachEvent('onclick', MyButtonClicked);" + 
                "}  " + 
                "function MyButtonClicked(){" + 
                    " window.external.notify('%%' + location.href);"+ 
                 "}" 
            }); 
        }
Here we are injecting javascript code that will add the event click to the "MyButton" defined in the html page, and each time the click event is faired the function "MyButtonClicked" wil called and it will fire the WebView's ScriptNotify event with the current url, but can be other value.

Then define the WebView's ScriptNotify event handler as following

C#
Edit|Remove
csharp 
 private void WebView_OnScriptNotify(object sender, NotifyEventArgs e)
{
   var url = e.Value;
}
 private void WebView_OnScriptNotify(object sender, NotifyEventArgs e) 
{ 
   var url = e.Value; 
}
 The event handler is fired and the url value is set.
